# 경기청년 메디케어 플러스

policy_id: youth-medicare-plus

source_type: policy_master

## 요약
건강검진 또는 예방접종 지원 대상 청년

## 대상 조건
나이: 19~39세 / 거주: 경기도 거주 청년 / 고용: 건강검진: 미취업청년 / 소득: 예방접종 신청자는 중위소득 120% 이하

## 신청 기간
26. 5. 20.(수) ~ 5.29.(금) 18:00

## 혜택
건강검진, 예방접종 비용 지원

## 지원 금액/방식
unknown

## 신청 방법
온라인 신청 > 서류 업로드 > 선정

## 제출 서류
- 신청서 (구분: required / 조건: 전체 신청자 / 발급처: 잡아바 어플라이 / 발급방법: 온라인 작성 / 발급시점: 신청기간 내 작성 / 형식: 온라인 제출 / 비고: 온라인 신청)
- 건강보험료 납부확인서 (구분: conditional / 조건: 예방접종 신청자 / 발급처: 국민건강보험공단 / 발급방법: 온라인 발급 / 발급시점: 공고일 이후 발급 권장 / 형식: PDF / 비고: 중위소득 확인용)
- 건강보험 자격확인서 (구분: conditional / 조건: 소득확인 필요자 / 발급처: 국민건강보험공단 / 발급방법: 온라인 발급 / 발급시점: 신청기간 내 제출 / 형식: PDF / 비고: 건강보험 가입 확인)
