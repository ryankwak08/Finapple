# 경기도 청년 면접수당

category: employment
source_type: policy_master
application_status: upcoming

## 요약
구직활동을 하는 경기도 거주 18세 이상 39세 이하 청년

## 대상 조건
나이: 18~39세 / 거주: 신청일 기준 주민등록상 경기도 내 거주 / 고용: 2026. 1. 1. 이후 일자리를 얻기 위한 면접에 응시한 청년

## 신청 기간
2026년 6-7월 1차 모집 예정

## 혜택
지역화폐 지원(연 최대 3회)

## 지원 금액/방식
신청일 기준 60일 이내 면접 1회 5만원 지역화폐 지원

## 신청 방법
https://apply.jobaba.net/bsns/bsnsListView.do#

## 제출 서류
- 신청서 (구분: required / 조건: 전체 신청자 / 발급처: 신청자 / 발급방법: 온라인 작성 / 발급시점: 신청 시 / 형식: 온라인 제출)
- 개인정보활용동의서 (구분: required / 조건: 전체 신청자 / 발급처: 신청자 / 발급방법: 온라인 작성 / 발급시점: 신청 시 / 형식: 온라인 제출)
- 채용공고문 (구분: required / 조건: 전체 신청자 / 발급처: 해당 회사 / 발급방법: 온라인 작성 / 발급시점: 신청 시 / 형식: PDF 또는 이미지)
- 주민등록등본 (구분: required / 조건: 전체 신청자 / 발급처: 정부 24 / 발급방법: 온라인 발급 / 발급시점: 신청 시)
- 면접확인서 (구분: required / 조건: 전체 신청자 / 발급처: 회사 담당자 작성 / 발급방법: 작성 후 파일 업로드 / 발급시점: 면접 진행 시 / 형식: PDF 또는 이미지 / 비고: 신청자 본인이 작성 및 날인(서명)하면 안됨/필요 시 면접 기업의 자체 양식 제출도 가능)
- 면접확인서대체서약서 (구분: optional / 조건: 면접 확인서 발급 불가한 경우 / 발급처: 본인 / 발급방법: 작성 후 파일 업로드 / 발급시점: 면접 진행 시 / 형식: PDF 또는 이미지 / 비고: 양식은 공고문 참조)

## 심사 및 지급
- 심사: 신청자격 확인>유사‧중복사업*참여여부 조회>온라인신청서,증빙서류 검증>선정/미선정
(잡아바어플라이 마이페이지 확인)>지역화폐 지급
(신청일부터 60일 내외)
- 결과 발표: 신청일 이내 60일
- 지급 일정: 신청일 이내 60일

## 주의사항
선정자가 지역화폐 어플리케이션 회원가입 및 이용등록을 완료해야 수당 지급 가능
