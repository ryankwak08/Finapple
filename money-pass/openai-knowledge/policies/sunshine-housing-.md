# 햇살하우징 사업

policy_id: sunshine-housing
category: housing
source_type: policy_master
application_status: always
data_confidence: medium

## 요약
에너지 효율 개선 및 주거환경 개선 지원

## 대상 조건
거주: 경기도 거주 저소득층 / 고용: 무관 / 소득: 기초생활수급자 및 차상위계층 중심 / 가구: 저소득 가구 / 주거: 노후주택 거주

## 신청 기간
시군별 상시 또는 연간 공고

## 혜택
단열·창호·보일러 교체 등 지원

## 지원 금액/방식
가구별 차등 지원

## 신청 방법
시군 신청 > 현장조사 > 대상자 선정 > 공사 진행

## 제출 서류
- 지원신청서 (구분: required / 조건: 전체 신청자 / 발급처: 지자체 / 발급방법: 방문 수령 / 발급시점: 신청기간 내 / 형식: PDF 또는 이미지 / 비고: 행정복지센터 제출)
- 주민등록등본 (구분: required / 조건: 전체 신청자 / 발급처: 정부 24 / 발급방법: 온라인 발급 / 발급시점: 공고일 이후 / 형식: PDF 또는 이미지 / 비고: 거주지 확인용)
- 소득증빙서류 (구분: required / 조건: 전체 신청자 / 발급처: 건강보험공단/복지로 / 발급방법: 온라인 발급 / 발급시점: 신청기간 내 / 형식: PDF / 비고: 중위소득 50% 이하 확인용)
- 임대차계약서 (구분: conditional / 조건: 임차 가구 / 발급처: 임대인/본인 / 발급방법: 사본 제출 / 발급시점: 신청 시 / 형식: PDF 또는 이미지 / 비고: 임차주택 거주자 제출)
- 주택 사진 (구분: conditional / 조건: 현장 확인 필요 시 / 발급처: 본인 / 발급방법: 직접 촬영 / 발급시점: 신청 시 / 형식: 이미지/PDF / 비고: 노후 상태 확인용)

## 심사 및 지급
- 심사: 현장점검 및 소득심사
- 결과 발표: 개별 통보
- 지급 일정: 공사 완료 후 집행

## 주의사항
예산 소진 시 조기 마감 가능

## 주요 탈락 사유
소득초과; 주택요건 미충족
