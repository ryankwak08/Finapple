## 요약
공고일 기준 경기도 기준 만 18-29세인 미취업/졸업 예정 청년

## 대상 조건
나이: 18~39세 / 거주: 공고일 기준 경기도 거주 / 고용: 공고일 기준 미취업

## 신청 기간
2026. 5. 27.(수) ~ 7. 21.(화) 17:00

## 혜택
청년지원사업 정보제공
- 1대1 전담 직업상담사 배정을 통한 진로설계, 취업컨설팅 등 실시                                                                                       - 취업지원 프로그램 운영(자기소개서 작성법, 면접노하우, 전·현직자 멘토링 등)
- 맞춤형 구인정보 제공 및 취업알선
- 취업자 사후관리(희망자에 한함)

## 지원 금액/방식
청년지원사업 정보제공
- 1대1 전담 직업상담사 배정을 통한 진로설계, 취업컨설팅 등 실시                                                                                       - 취업지원 프로그램 운영(자기소개서 작성법, 면접노하우, 전·현직자 멘토링 등)
- 맞춤형 구인정보 제공 및 취업알선
- 취업자 사후관리(희망자에 한함)

## 신청 방법
https://apply.jobaba.net/bsns/bsnsDetailView.do?bsnsSeq=5606

## 제출 서류
- 신청서 (구분: required / 조건: 전체 신청자 / 발급처: 본인 / 발급방법: 통합접수시스템을 통한 직접 접수 / 발급시점: 신청 시 / 형식: 온라인 제출)
- 개인정보 수집/이용/제공 동의서 (구분: required / 조건: 전체 신청자 / 발급처: 본인 / 발급방법: 통합접수시스템을 통한 직접 접수 / 발급시점: 신청 시 / 형식: 온라인 제출)
- 주민등록표 초본 (구분: required / 조건: 전체 신청자 / 발급처: 정부24 / 발급방법: 파일 업로드 후 마이데이터로 제출 가능 / 발급시점: 공고일 이후 발급 / 형식: PDF 또는 이미지 / 비고: 주민등록번호 뒷자리 포함하여 발급 필수)
- 고등학교 또는 대학교 졸업(예정)증명서 (구분: conditional / 조건: 우선선발 대상 / 발급처: 졸업 예정 학교 / 발급방법: 온라인 제출 / 발급시점: 신청기간 내 / 형식: PDF 또는 이미지 / 비고: 공고일 기준 1년 이내 졸업자 또는 졸업예정자에 한함)
- 병역사항이 포함된 주민등록표 초본 (구분: optional / 조건: 병역 이행자 대상 / 발급처: 정부24 / 발급방법: 파일 업로드 후 마이데이터로 제출 가능 / 발급시점: 공고일 이후 발급 / 형식: 온라인 제출 / 비고: 병역 의무를 이행한 사람은 병역 기간만큼 신청기간 연장 가능)

## 심사 및 지급
- 심사: 신청서 접수>서류검증>합격자 발표> 상담 및 프로그램 운영
- 결과 발표: 접수 후 4주 이내 발표
- 지급 일정: 선발 후 3개월간 진행

## 주의사항
확인 필요

## 주요 탈락 사유
- 고등학교 졸업 후 대학교에 재학 중이면서 고교 졸업증명서를 제출해 우선선발을 노린 경우 (적발 시 선발 취소)                                                  - 취업계획서 문항에 100자 미만으로 불성실하게 답변한 경우
