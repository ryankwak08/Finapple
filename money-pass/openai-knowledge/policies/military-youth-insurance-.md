# 군복무 경기청년 상해보험 가입지원

## 요약
군복무 중인 경기도 청년

## 대상 조건
거주: 경기도 주민등록 청년 / 고용: 현역 군복무 / 소득: 없음

## 신청 기간
2026.1.1 ~ 2026.12.31

## 혜택
상해보험 자동가입 및 보험금 지원

## 지원 금액/방식
보장항목별 상이

## 신청 방법
사고발생 > 보험금 청구 > 심사

## 제출 서류
- 보험금 청구서 (구분: required / 조건: 보험금 청구자 / 발급처: 보험사 / 발급방법: 다운로드 후 작성 / 발급시점: 사고 발생 후 제출 / 형식: PDF / 비고: 보험사 양식 사용)
- 사고증빙서류 (구분: required / 조건: 사고 발생자 / 발급처: 병원/군부대 / 발급방법: 기관 발급 / 발급시점: 사고 이후 제출 / 형식: PDF / 비고: 진단서, 입퇴원확인서 등)
- 주민등록초본 (구분: conditional / 조건: 경기도 거주 확인 필요 시 / 발급처: 정부24 / 발급방법: 온라인 발급 / 발급시점: 신청일 기준 최신 / 형식: PDF / 비고: 주민등록 기준 확인)

## 심사 및 지급
- 심사: 보험사 심사
- 결과 발표: unknown
- 지급 일정: 사고 승인 후 지급

## 주의사항
자동가입이지만 보험금은 직접 청구

## 주요 탈락 사유
