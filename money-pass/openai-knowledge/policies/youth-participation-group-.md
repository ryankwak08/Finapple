# 경기도 청년참여기구 운영

policy_id: youth-participation-group
category: welfare
source_type: policy_master
application_status: closed
data_confidence: medium

## 요약
경기도 청년 정책참여 희망자

## 대상 조건
나이: 19~39세 / 거주: 경기도 거주 또는 경기도 소재 대학, 직장 / 고용: 무관 / 소득: 없음

## 신청 기간
2025.8.18 ~ 2025.8.24

## 혜택
청년정책 제안 및 참여활동 지원

## 지원 금액/방식
unknown

## 신청 방법
온라인 신청 > 서류 검토 > 선정

## 제출 서류
- 지원신청서 (구분: required / 조건: 전체 신청자 / 발급처: 잡아바 어플라이 / 발급방법: 온라인 작성 / 발급시점: 신청기간 내 작성 / 형식: 온라인 제출 / 비고: 온라인 신청서 작성)
- 자기소개서 (구분: required / 조건: 전체 신청자 / 발급처: 신청자 작성 / 발급방법: 파일 업로드 / 발급시점: 신청기간 내 제출 / 형식: PDF / 비고: 활동경험 및 지원동기 작성)
- 재학증명서 (구분: conditional / 조건: 경기도 소재 대학 재학생 / 발급처: 학교 / 발급방법: 온라인 발급 / 발급시점: 신청기간 내 제출 / 형식: PDF / 비고: 대학 재학 여부 증빙)
- 재직증명서 (구분: conditional / 조건: 경기도 소재 직장 재직자 / 발급처: 회사 / 발급방법: 회사 발급 / 발급시점: 신청기간 내 제출 / 형식: PDF / 비고: 경기도 소재 직장 증빙)

## 심사 및 지급
- 심사: 서류심사
- 결과 발표: unknown
- 지급 일정: unknown

## 주의사항
신청기한 이후 접수 불가

## 주요 탈락 사유
자격요건 미충족
