# 경기도 취약계층 주택화재 안심보험 지원

policy_id: housing-fire-insurance
category: housing
source_type: policy_master
application_status: always
data_confidence: medium

## 요약
화재 피해 대비 보험 지원

## 대상 조건
거주: 경기도 취약계층 주민 / 고용: 무관 / 소득: 기초생활수급자 등 취약계층 중심 / 가구: 취약계층 가구 / 주거: 주택 거주자

## 신청 기간
연중 운영

## 혜택
화재·폭발·붕괴 피해 보험 보장

## 지원 금액/방식
지자체별 상이

## 신청 방법
별도 신청 없음

## 제출 서류
- 주민등록등본 (구분: conditional / 조건: 지자체 신청형 사업일 경우 / 발급처: 정부24 / 발급방법: 온라인 발급 / 발급시점: 신청 시 / 형식: PDF / 비고: 일부 시군만 신청 필요)
- 기초생활수급자 증명서 (구분: conditional / 조건: 수급자 대상 / 발급처: 주민센터/정부24 / 발급방법: 온라인 발급 / 발급시점: 신청 시 / 형식: PDF / 비고: 취약계층 확인용)
- 차상위계층 확인서 (구분: conditional / 조건: 차상위 대상자 / 발급처: 주민센터 / 발급방법: 온라인 /방문 / 발급시점: 신청 시 / 형식: PDF / 비고: 해당자만 제출)

## 심사 및 지급
- 심사: 지자체 대상자 확인
- 결과 발표: 자동 적용
- 지급 일정: 사고 발생 시 보험금 지급

## 주의사항
지자체별 보장 범위 상이

## 주요 탈락 사유
대상자 기준 미충족
