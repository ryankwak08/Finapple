# 경기청년 갭이어 프로그램

policy_id: youth-gap-year
category: education
source_type: policy_master
application_status: closed
data_confidence: medium

## 요약
경기도 청년 자기탐색 프로그램 참여 희망자

## 대상 조건
나이: 19~39세 / 거주: 경기도 거주 청년 / 고용: 무관 / 소득: unknown

## 신청 기간
공식 공고문 확인 필요

## 혜택
프로젝트, 멘토링, 활동 지원

## 지원 금액/방식
unknown

## 신청 방법
잡아바 접속 > 신청서 작성 > 서류 제출 > 심사

## 제출 서류
- 신청서 (구분: required / 조건: 전체 신청자 / 발급처: 잡아바 어플라이 / 발급방법: 온라인 작성 / 발급시점: 신청기간 내 작성 / 형식: 온라인 제출 / 비고: 공식 신청페이지 작성)
- 활동계획서 (구분: required / 조건: 전체 신청자 / 발급처: 신청자 작성 / 발급방법: 파일 업로드 / 발급시점: 신청기간 내 제출 / 형식: PDF / 비고: 자기소개 및 활동계획 포함 가능)
- 주민등록 초본 (구분: conditional / 조건: 거주요건 확인 필요 시 / 발급처: 정부24 / 발급방법: 온라인 발급 / 발급시점: 공고일 이후 발급 권장 / 형식: PDF / 비고: 주소이력 포함 여부 확인 필요)

## 심사 및 지급
- 심사: 서류심사 후 선발
- 결과 발표: unknown
- 지급 일정: unknown

## 주의사항
제출서류 누락 시 심사 제외 가능

## 주요 탈락 사유
거주요건 미충족
